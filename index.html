1. git 是一个源代码管理工具 早期是linux 之父用来管理linux系统源代码的工具 后来工具公开了 别人发掘这个工具很好用并且得到推广 2. github:是一个git的源代码管理服务的提供商（我们项目的源代码远程仓库 都是在github服务器上）专门放仓库和代码的 同时因为很多开源的项目都在github服务器上 实现了一个项目交流平台 程序代码的交流平台 程序员可以像分享朋友圈一样 去分享自己的代码 3. 区分一下 git和github 他们不是一个东西 也不属于一个公司 1. git是一个工具 是一种技术 一种语言 2. github是一个服务器 平台 3. 但是我们开发 需要学会git工具使用 也要学会github使用 git的代码都要提交到github 4. 使用前提 1. 安装git工具 （git软件） 1. 打开day01 》 教学资料 Git-2.13.0-32-bit 找到对应的位数安装 2. 双击打开一路next不要修改目录 （不能安装在中文目录下） 3. 安装完毕之后检测是否成功在桌面 右键 git bash here 打开这个是一个黑窗就对了 2. 注册一个github账号 1. 打开 https://github.com/ 网站 2. 点击右上角的 Sign up 或者注册 3. 输入用户名 和邮箱 和 密码 点击create a count 4. 不要勾选复选框 直接点击continue 5. 选择你的个人信息 和意向 随便选 点击submit提交 6. 点击start a project 7. 登录你的邮箱去验证一下邮箱 8. 如果出现create a new respsitory 就表示邮箱验证成功 可以使用了 5. git的使用流程 1. 在本地有一个项目源代码 2. 在本地有一个专门用来管理源代码的仓库 3. 在github服务器上有一个远程的仓库 4. 代码是先从本地源代码提交到本地的仓库 再把本地仓库的代码提交到服务器仓库 6. 使用git的步骤 1. 在本地创建一个需要被管理的项目 2. 在本地的项目里面去创建一个git本地仓库 （管理你需要提交的源代码）在当前项目打开git黑窗 执行git init 回车 3. 去github上去创建一个远程的仓库 1. 进入github登录账号 2. 点击右上角的头像 点击your profile 3. 点击 Repositories 4. 点击右侧new 5. 输入 Repository name 项目的仓库名 不要用中文 还可以输入一些描述 描述可以有中文 6. 点击create respository
